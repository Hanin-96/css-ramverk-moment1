Om uppgiften

Uppgiften gick ut på att bygga upp en webbplats med flera undersidor, där varje undersida har en specifik ramverk. Jag valde ramverken Bootstrap och Tailwind. Jag försökte skapa undersidor med liknande styling för att märka större skillander under utvecklingen. Undersidorna innehåller för/nackdelar, länk till ramverket, formulär och en tabell. Startsidan och publiceringssidan visar hur jag vanligtvis hade stylat CSS med en extern stylesheet.


Bootstrap var mycket enklare att sätta upp snabbt och hade många inbyggda färdiga klasser som snabbt kunde bygga upp komponenter såsom navbar, table och form. Det är även mycket smidigare att lägga till ikoner då bootstrap innehåller stort bibliotek med ikoner att välja bland och inkludera med klasserna. Där sparade jag en massa tid. Tailwind däremot krävde flera klasser och justeringar för att uppnå liknande resultat. Dock kändes det tryggade och smidigare att jobba i Tailwind. Jag gillar generellt att jobba mycket med CSS styling och därmed ha mer kontroll över stylingen. Det är precis det Tailwind erbjuder.


Jag ser charmen i att jobba med Bootstrap om man inte gillar CSS eller vill hitta snabba genvägar för snabbare resultat. Personligen tror jag mer på Tailwind som ramverk, då den både stylar elementen men också lämnar mycket plats för utvecklaren att anpassa webbplatsen efter sina behov/syfte.
