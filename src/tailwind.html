Länk till Tailwind

Fördelar

  • Tar längre tid än Bootstrap att sätta upp.
  • Mycket mindre CSS kod, mer beroende av klasser.
  • Responsiv och anpassningsbar, utan behov av media queries i CSS.
  • Mer kontroll över stylingen.
  • Fördelaktig för att applicera responsivitet.
  • Mer inbyggda färgmöjligheter
  • Enklare att skriva över CSS kod med hjälp av config.
  • Mer lättanvänt och flexibel.
  • Större möjligheter att bygga upp mer unika och personliga webbplatser.
  • Mycket tydligare dokumentation.

Nackdelar

  • Behövs mycket mer klasser.
  • Tar längre tid än Bootstrap att sätta upp.
  • Kan bli många klasser under samma rad.
  • Många klassnamn att memorera, behövs ständig kontroll med dokumentation.
  • Tar tid att lära sig arbetssättet.
  • Mindre flexibelt för att göra större ändringar.
  • Många av deras färdiga komponentlösningar med klasser kostar pengar, därför körde jag enbart med enkla klasser.
